
* DX Tools *

DX should build cleanly with the following tools (this is mostly FC15).  Other
tools or tool versions may also work, but are probably untested.

Note that the toolchain must be capable of generating 32b, ELF binaries.
Cross-versions of these tools will be necessary if the native tools cannot
produce this file format (e.g., building under cygwin, or on a 64b host).
See cross_compiling.txt, if necessary.


* GNU bash, version 4.2.10(1)-release (x86_64-redhat-linux-gnu)

* gcc version 4.6.1

* GNU assembler (GNU Binutils) 2.21.1

* GNU ld (GNU Binutils) 2.21.1

* GNU Make version 3.82

* debugfs 1.41.14 (22-Dec-2010)

* genisoimage 1.1.11 (Linux)

* Doxygen version 1.7.4
